= Laboratorio 2: Nuestro primer agente

'Vamos 
Un agente de IA es un sistema que puede tomar decisiones o responder preguntas de forma inteligente. En este caso, es un asistente conversacional que usa inteligencia artificial para ayudarte a encontrar información en documentos.

En este segundo laboratorio, vas a crear y probar tu primer agente. Este agente podrá responder preguntas usando documentos reales (como PDFs, presentaciones, manuales o páginas web) gracias a una técnica llamada RAG (Retrieval-Augmented Generation).
